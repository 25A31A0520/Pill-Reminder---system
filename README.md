# Pill Reminder System 💊

## Overview

The Pill Reminder System is a C++ console-based application designed to help users manage their medicine schedules efficiently. The program allows users to store and manage medicine reminders using a linked list data structure.

## Features

* Add a new pill reminder
* View all reminders
* Delete a reminder
* Simple menu-driven interface
* Uses Linked List from Data Structures

## Technologies Used

* C++
* Data Structures (Linked List)
* Dev-C++

## How It Works

The program stores each reminder as a node in a linked list. Each node contains information about the medicine name, dosage, and reminder time. Users can add, view, or delete reminders through a simple console menu.

## Future Improvements

* Sound notification for reminders
* Time-based automatic alerts
* Mobile or GUI version of the application
